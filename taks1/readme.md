
# Task 1: File Management System

## Project Overview
A C++ console application that performs file operations including writing, appending, and reading text files using standard `<fstream>` library streams.

## Features
- Interactive menu-driven interface
- Support for multi-line text input with "END" as termination marker
- Safe file handling with error checks

## Technologies
- C++ (C++11/14)
- `<iostream>`, `<fstream>`, `<string>`

## Setup and Usage
1. Compile: `g++ -std=c++14 task1.cpp -o file_management`
2. Run: `./file_management`
3. Use menu options to write, append, or read from `data.txt`
