# Task 4: Basic Arithmetic Expression Compiler

## Project Overview
A simple compiler program that parses and evaluates basic arithmetic expressions input by the user.

## Features
- Recursive parsing of expressions
- Validation and error handling
- Outputs calculated expression results

## Technologies
- C++11/14
- `<iostream>`, `<string>`

## Setup and Usage
1. Compile: `g++ -std=c++14 task4.cpp -o expression_compiler`
2. Run: `./expression_compiler`
3. Enter arithmetic expressions to evaluate
