# Task 3: Graphical Snake Game

## Project Overview
A graphical snake game implemented using SFML libraries, featuring increasing difficulty and sound effects.

## Features
- Real-time input handling and rendering
- Dynamic difficulty adjustment
- Sound effects for game actions

## Technologies
- C++ with SFML libraries
- `<iostream>`, SFML Graphics and Audio modules

## Setup and Usage
1. Install SFML libraries
2. Compile: `g++ -std=c++14 task3.cpp -o snake_game -lsfml-graphics -lsfml-window -lsfml-audio -lsfml-system`
3. Run: `./snake_game`
